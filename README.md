## What the app does

After donating blood in Latvia, you can get a Narvesen coffee coupon via a link and an e-mail
address.  
Normally it works like this:

- You scan a QR code after donation.
- You enter an e-mail.
- The system sends you a coupon.
- Each e-mail usually gets **one specific coupon**, and reusing the same e-mail will usually resend
  the same coupon.

This app:

- Automates the process of **getting a new coupon** after donation.
- Stores your coupons locally (available / used).
- Helps you not lose the coupon and quickly find it later.
- Has some helpers for working with e-mails (including Gmail aliases).

The main goal is to remove repetitive manual steps for a **frequent donor** and to have some fun
writing Dart/Flutter code.

## Setup & Run on Android

### 1. Install dependencies

From the project root:

```bash
flutter pub get
```

### 2. Generate JSON serialization code

The project uses `json_serializable` / `build_runner` for `*.g.dart` files.

Generate once:

```bash
dart run build_runner build --delete-conflicting-outputs
```

### 3. Generate launcher icons & splash screen

Icons are generated by `flutter_launcher_icons`, splash by `flutter_native_splash`.

**Generate launcher icons:**

```bash
dart run flutter_launcher_icons
```

**Generate native splash:**

```bash
dart run flutter_native_splash:create
```

Run these again if you change the icon or splash assets/config.

### 4. Run on device / emulator (debug)

Connect an Android device with USB debugging enabled or start an emulator, then:

```bash
flutter run
```

This builds and launches a debug build of the app.

## Build a release APK (Android)

1. (Optionally) configure signing for a proper release build:
   Flutter
   docs: [https://docs.flutter.dev/deployment/android](https://docs.flutter.dev/deployment/android)

   For quick local use you can also just install the unsigned debug APK on your own device.

2. Build a release APK:

   ```bash
   flutter build apk --release
   ```

3. The output will be here:

   ```text
   build/app/outputs/flutter-apk/app-release.apk
   ```

4. Copy/install this APK to your phone (ADB, file transfer, etc.) and install it manually.
